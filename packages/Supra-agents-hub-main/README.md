# Supra Agents Hub

## What is Supra?  

Supra is a **high-performance Layer 1**, starting with MoveVM, designed to enable Smart contracts with **instant execution** and **MultiVM operability**. It achieved **500K TPS on 300 nodes** with **sub-second consensus latency**, making it the **best environment for AI Agents** in DeFi and beyond.

## Why Choose Supra for AI Agents?  

Supra is built for **autonomous execution** and **real-time data processing**, enabling AI agents to make **instant decisions** and **execute smart contracts** natively. Key benefits include:

- **Ultra-fast Transactions** ‚Üí 500K+ TPS ensures AI Agents execute **before competitors**  
- **Native Automation** ‚Üí AI-driven contract execution **without external dependencies**  
- **Real-time Market Data** ‚Üí Integrated **Oracle price feeds** for live insights  
- **Multi-chain AI Compatibility** ‚Üí Built-in **Bridgeless Cross Chain Communication Protocol/SupraNova & OpenBlocks.ai** for cross-chain AI automation  
- **AI-ready On-Chain Strategies** ‚Üí Supports **AutoFi** (Automatic DeFi) for **AI-managed portfolios**
